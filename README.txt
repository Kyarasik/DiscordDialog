ПЕРЕД ИСПОЛЬЗОВАНИЕМ БОТА ВНИМАТЕЛЬНО ПРОЧИТАЙТЕ РУКОВОДСТВО

HTTP-запросы с Discord обрабатываются библиотекой Sleepy-discord (yourWaifu)
Руководство по установке всех нужных файлов для работы сервера: https://github.com/yourWaifu/sleepy-discord (Раздел Requirements)
Так же на официальной странице библиотеки вы можете найти синтаксис данной библиотеки для своих проектов
Данный проект разчитан на macos устройства

КАК ЗАПУСТИТЬ БОТА:
Убедитесь что все зависимости библиотеки Sleepy Discord у вас установлены и проверены

1. Создайте проект в любом конструкторе кода и назовите его "minion" (Если хотите сменить название нужно пересобрать сборку, чуть ниже об этом есть руководство)
2. В исполняемом файле source.cpp вставьте токен вашего дискорд-бота, как создать бота и как найти его токен вы можете найти в любых источниках информации либо на сайте https://discord.com/developers/applications
3. В терминале вашего компьютера (Или компилятора) перейдите в папку build, на macos это можно сделать cd путь_до_папки
4. Обновите испольняемый файл до последней версии командой make (На других платформах команда может отличаться)
5. После чего введите в консоль ./название_вашего_проекта, после чего в терминале будет сообщение о успешном подключении
После этого ваш бот будеть работать согласно данному коду

КАК ПЕРЕСОБРАТЬ ПРОЕКТ:
Если вы хотите пересобрать проект чтоб поменять его имя либо для иных причин
1. Удалите содержимое папок build и deps 
2. В терминале вашего компьютера (Или компилятора) перейдите в папку deps, на macos это можно сделать cd путь_до_папки
3. Введите команду git clone https://github.com/yourWaifu/sleepy-discord (Для этого вам понадобиться расширение git либо вы можете установить вручную с сайта в архив)
4. Вернитесь в папку проекта командой cd .. (если у вас macos)
5. Не забудьте в файле CMakeLists.txt переименовать все строчки с minion на ваше название проекта (Также и другие зависимости если вы еще что-то меняли)
6. В терминале вашего компьютера (Или компилятора) перейдите в папку build, на macos это можно сделать cd путь_до_папки
7. Введите команду cmake путь_до_вашего_проекта должен начаться длительный процесс установки сборки
8. Обновите испольняемый файл до последней версии командой make (На других платформах команда может отличаться)
9. После чего введите в консоль ./название_вашего_проекта, после чего в терминале будет сообщение о успешном подключении
После этого ваш бот будеть работать согласно данному коду
